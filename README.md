# Music Downloader

Python Full Stack Developer | Flask | HTML | CSS | SQLite

---

## Description
This is a simple Music Downloader app built using Python and Flask.  
It allows users to search and download royalty-free music using open APIs, with a user-friendly interface.

## Features
- Search for music tracks
- Download music locally
- User-friendly interface
- Stores data using SQLite

## Technologies Used
- Python
- Flask
- HTML, CSS
- SQLite
